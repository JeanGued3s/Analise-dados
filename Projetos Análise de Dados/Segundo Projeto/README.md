Após as etapas de coleta, tratamento e análise dos dados, organizei as informações:

- lucro geral;
- lucro por país; e
- quantidade de produtos em estoque.

De forma complementar, realizei a análise inversa, identificando:

- total de perdas;
- perdas por país;
- quantidade de produtos em falta no estoque.

O objetivo do projeto foi fornecer uma visão clara dos resultados financeiros da empresa, permitindo identificar oportunidades de melhoria e apoiar a tomada de decisão baseada em dados.

Meu assessoramento séria reorganizar o estoque tendo em vista a falta de muitos produtos. Repondo aos poucos para ver qual produto 
sairia/venderia mais e após isso fazer um pente fino para reduzir quantidade para trabalhar com qualidade.
