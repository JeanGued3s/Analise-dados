Análise de Dados - Power BI

Hoje finalizo um projeto de análise de dados que venho desenvolvendo com o objetivo de aprimorar minhas habilidades em Power BI e Análise de Dados.

A base de dados utilizada foi uma tabela disponível no https://www.kaggle.com, o que me permitiu trabalhar com dados relevantes em um ambiente corporativo de aprendizado, reforçando boas práticas de análise, modelagem e visualização de dados.

O objetivo foi transformar os dados brutos em informações claras e visuais.

Desenvolvi no projeto:
- tratamento e modelagem de dados;
- criação de kpi lucro total;
- modo de pagamento;
- lucro ao longo do tempo;
- estados com maior lucro; e
- categória das vendas e as sub-categórias.
